  1. ol(Ordered List) - 정렬된 순서로 목록을 정의하는 태그입니다.
  2. ol태그의 속성으로 type, reversed, start가 존재합니다.
    1. type은 항목이 메겨지는 번호의 유형을 설정하는 속성입니다.
    2. reversed는 항목을 역순으로 설정하는 속성입니다.
    3. start는 항목에 매겨지는 번호의 시작 값을 정하는 속성입니다.
  3. ul(Unordered List) - 순서의 상관없이 목록을 정의하는 태그입니다.
  4. li(List Item) - 항목을 설정하는 태그입니다.
