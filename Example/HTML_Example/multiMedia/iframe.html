iframe

다른 HTML 페이지를 현재 페이지에 삽입할 때 사용하는 태그입니다.

name
프레임의 이름을 지정해주는 속성입니다.
src
포함할 문서의 URL을 지정해주는 속성입니다.
width, height
프레임의 가로, 세로 너비를 지정해주는 속성입니다.
allowfullscreen
전체 화면 모드 사용 여부를 지정해주는 속성입니다.
frameborder
프레임 테두리 사용 여부를 지정해주는 속성입니다.
sandbox
보안을 위한 읽기 전용으로 삽입하는 속성입니다.
  • allow-form : 양식 제출 가능
  • allow-scripts : 스크립트 실행 가능
  • allow-same-origin : 같은 출처(도메인)의 리소스 사용 가능
