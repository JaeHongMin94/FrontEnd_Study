video

동영상 콘텐츠(mp4)를 삽입할 때 사용하는 태그입니다.

autoplay
준비되면 바로 재생하는 속성입니다.
controls
audio태그의 제어 메뉴를 표시해주는 속성입니다.
loop
재생이 끝났을 경우 다시 처음부터 재생시켜주는 속성입니다.
muted
음소거 여부를 지정해주는 속성입니다.
poster
동영상 썸네일 이미지의 경로를 설정해주는 속성입니다.
preload
페이지가 로드될 때 파일을 로드할지를 지정해주는 속성입니다.
  • none : 로드하지 않습니다.
  • metadate : 메타데이터만 로드합니다.
  • auto : 전체 파일을 로드합니다.
src
콘텐츠의 URL을 지정해주는 속성입니다.
width, height
동영상의 가로, 세로 너비를 지정해주는 속성입니다.
