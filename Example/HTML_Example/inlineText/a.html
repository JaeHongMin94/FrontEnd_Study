a태그의 속성

  • download

    이 요소가 리소스를 다운로드하는 용도로 사용되는것을 명시하는 속성입니다.
    파일을 올리고 download속성을 작성하지 않는다면 현재창에 파일에 대한 정보가 표시됩니다.

  • href

    링크 URL을 걸어주는 속성입니다.(하이퍼링크)

  • hreflang

    링크 URL의 언어를 설정해주는 태그입니다. 자주 사용하는 속성은 아닙니다.

  • rel

    현재 문서와 링크 URL의 관계를 명시해주는 속성입니다.

  • target

    링크 URL의 표시 위치를 명시해주는 속성입니다.
    _self(현재 창에 웹페이지를 불러옵니다.), _blank(새로운 창에 웹페이지를 불러옵니다.)
