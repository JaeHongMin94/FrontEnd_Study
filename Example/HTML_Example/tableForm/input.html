input태그

사용자에게 입력 받을 데이터 양식 태그입니다.

autocomplete
사용자가 이전에 입력한 값으로 자동 완성 기능을 사용할 것인지 여부
autofocus
페이지가 로드될 때 자동으로 포커스
checked
양식이 선택되었음을 표시
disabled
양식을 비활성화
form
form의 id 속성 값
list
참조할 datalist의 id 속성 값
max
지정 가능한 최대 값
min
지정 가능한 최소 값
maxlength
입력 가능한 최대 문자 수
multiple
둘 이상의 값을 입력 할 수 있는지 여부
name
양식의 이름
placeholder
사용자가 입력할 값의 힌트
readonly
수정 불가한 읽기 전용
step
유효한 증감 숫자의 간격
type
입력 받을 데이터의 종류
value
입력 받을 데이터의 종류
