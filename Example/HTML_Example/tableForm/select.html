select태그

옵션을 선택하는 메뉴

autocomplete
사용자가 이전에 입력한 값으로 자동 완성 기능을 사용할 것인지 여부
disabled
선택 메뉴를 비활성화
form
선택 메뉴가 속할 하나 이상의 form의 id속성 값
multiple
다중 선택 여부
name
선택 메뉴의 이름
size
한 번에 볼 수 있는 행의 개수
option태그

선택 메뉴(select)나 자동완성(datalist)에서 사용될 옵션

label
표시될 옵션의 제목
disabled
옵션을 비활성화
selected
옵션이 선택되었음을 표시
value
양식으로 제출될 값
optgroup태그

option을 그룹화

label
(필수)옵션 그룹의 이름
disabled
옵션 그룹을 비활성화
datalist태그

input에 미리 정의된 옵션을 지정하여 자동완성(Autocomplete) 기능을 제공하는 데 사용
input의 list 속성 바인딩.
option을 포함하여 정의된 옵션을 지정.
