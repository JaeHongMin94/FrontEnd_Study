값 의미 기본값
transition-property 전환 효과를 사용할 속성 이름 all
transition-duration 전환 효과의 지속시간 설정 0s
transition-timing-function 타이밍 함수(애니메이션 전환 효과를 계산하는 방법) 지정 ease
transition-delay 전환 효과의 대기시간 설정 0s
