float

요소를 좌우 방향으로 띄움(수평 정렬)

float 해제
float 속성이 적용된 요소의 주위로 다른 요소들이 흐르게 되는데 이를 방지하기 위해 속성을 해제 해야함
1. 형제요소에 clear: (left, right, both) 추가하여 해제
2. 부모요소에 overflow: (hideen, auto) 추가하여 해제
3. 부모요소에 clearfix 클래스 추가하여 해제(추천!)

값 의미
none 요소 띄움 없음
left 왼쪽으로 띄움
right 오른쪽으로 띄움
형제 요소에 clear
부모요소에 overflow
부모요소에 clearfix 클래스
