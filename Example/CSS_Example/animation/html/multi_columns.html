columns 다단을 정의
값 의미 기본값
auto 브라우저가 단의 너비와 개수를 설정 auto
colum-width 단의 최적 너비를 성정 auto
colum-count 단의 개수를 설정 auto
columns-gap 단과 단사이의 간격 설정
값 의미
normal 브라우저가 단과 단 사이의 간격을 설정(기본값)
단위 px, em, cm등 단위로 지정
columns-rule 단과 단 사이의(구분) 선을 지정
값 의미 기본값
column-width 선의 두께를 지정 medium
colum-style 선의 종류를 지정 none
colum-color 선의 색상을 지정 요소의 글자색과 동일
